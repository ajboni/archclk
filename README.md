# archclk -Arduino Chess Clock

A simple but functional chess clock made with arduino nano and a few components.

## Features

- 12 Game modes.
- Game mode select, pause, and reboot.
- Win/Loose state.
- Current turn indicator.

## Game modes.

	- Bullet 1 + 0
	- Bullet 2 + 1
	- Blitz 3 + 0
	- Blitz 3 + 2
	- Blitz 5 + 0
	- Blitz 5 + 3
	- Rapid 10 + 0
	- Rapid 10 + 5
	- Rapid 15 + 10
    - Classical 30 + 0
    - Classical 30 + 20
    - Classical 60 + 0
